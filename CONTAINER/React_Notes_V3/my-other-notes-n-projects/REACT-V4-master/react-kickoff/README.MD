React js Kickoff
================
This course is designed to introduce JavaScript developers to the React Library.

Topics
------
* Introducing React
* Introducing JSX
* Properties
* Challenge : Star Component
* Events, State, Refs
* Challenge : Dot Component
* Nesting Components
* Finishing the Bulletin Board
* Challenge : Star Ratings
