Enter the Hexadecimal (Hex) Color code in the first box and click the Button. You will see the result in RGB (red, green, blue) code in the second box. For example, enter #FF0000 to get color red. Usually, a Hex color code consists of a combination of six-digit values. However, you can use Shorthand Hex codes in your web page, particularly in CSS. Therefore, the shorthand Hex code of #FF0000 is F00.
